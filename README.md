## 🧠 Sobre este proyecto

Este repositorio contiene mi portafolio personal como estudiante de **Ingeniería de Software**, donde documento:

* Mi progreso como desarrollador
* Mis proyectos
* Mis experimentos
* Mi visión a largo plazo como futuro científico en computación

Este sitio no es solo una web… es una **bitácora técnica y personal**.

---

## 🧭 Estructura del proyecto

El sitio está organizado en múltiples páginas, cada una con un propósito claro:

```
/
├── index.html       # Página principal (entrada al sitio)
├── about.html       # Sobre mí (historia, filosofía, visión)
├── projects.html    # Proyectos detallados
├── lab.html         # Experimentos y pruebas técnicas
├── blog.html        # Bitácora de aprendizaje
├── roadmap.html     # Ruta de aprendizaje y objetivos
├── team.html        # The KernelGarden (equipo)
├── contact.html     # Información de contacto
├── style.css        # Estilos globales
└── assets/          # Imágenes, recursos y media
```

## 🧪 Desarrollo local

Para correr el proyecto en local:

```bash
python -m http.server 8080
```

Luego abre:

```
http://localhost:8080
```
